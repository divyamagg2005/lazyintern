# Auto-Approval: No Score Threshold

## ✅ Change Made

**REMOVED** score threshold from auto-approval system.

### Before
```python
AUTO_APPROVE_THRESHOLD = 90  # Only drafts with score >= 90%

if full_score >= AUTO_APPROVE_THRESHOLD:
    auto_approve()
```

### After
```python
# No threshold - ALL drafts auto-approved after 2h

for draft in pending_drafts:
    auto_approve()  # No score check
```

---

## 🎯 New Behavior

### ALL Leads Get Emails

**Every lead that makes it through the pipeline will receive an email:**

1. ✅ Lead discovered and scored
2. ✅ Email extracted (regex or Hunter)
3. ✅ Email validated (or not - you send to unverified too)
4. ✅ Draft generated by Groq
5. ✅ SMS sent to you for approval
6. ⏳ Wait 2 hours...
7. ✅ **Auto-approved (regardless of score)**
8. ✅ Email sent (after random delay)

### Score Still Used For

The scoring system is still active and useful for:

- **Pre-scoring (Gate 1)**: Filters out low-quality jobs before email extraction
  - Score < 40 → Marked as 'low_priority', skipped
  
- **Full scoring (Gate 3)**: Filters out low-quality matches before drafting
  - Score < 60 → Marked as 'low_priority', skipped

- **Logging & Analytics**: Track which leads are high vs low quality
  - Helps you understand what's working
  - Can adjust scoring weights later

- **Manual Review**: You see the score in SMS
  - "LazyIntern (75%) AI Intern at Company..."
  - Helps you decide YES/NO faster

**But NOT for auto-approval** - All drafts get auto-approved after 2h

## 🔍 Monitor It

### Check Auto-Approvals
```sql
-- All auto-approved drafts (regardless of score)
SELECT 
    ed.id,
    ed.approved_at,
    i.full_score,
    i.company,
    i.role
FROM email_drafts ed
JOIN leads l ON ed.lead_id = l.id
JOIN internships i ON l.internship_id = i.id
WHERE ed.status = 'auto_approved'
ORDER BY ed.approved_at DESC;
```

### Check Score Distribution
```sql
-- See what scores are being auto-approved
SELECT 
    CASE 
        WHEN i.full_score >= 90 THEN '90-100 (High)'
        WHEN i.full_score >= 75 THEN '75-89 (Medium)'
        WHEN i.full_score >= 60 THEN '60-74 (Low)'
        ELSE '<60 (Very Low)'
    END as score_range,
    COUNT(*) as count
FROM email_drafts ed
JOIN leads l ON ed.lead_id = l.id
JOIN internships i ON l.internship_id = i.id
WHERE ed.status IN ('auto_approved', 'sent')
GROUP BY score_range
ORDER BY score_range DESC;
```
