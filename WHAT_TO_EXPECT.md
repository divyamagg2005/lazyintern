## 🔧 Common Issues & Fixes

### Issue 1: No Internships Discovered

**Symptoms:**
- Dashboard shows 0 internships
- Scheduler logs show "Discovery complete: 0 sources scraped"

**Causes:**
- All sources skipped due to frequency tracking
- Scraping failed for all sources
- Network issues

**Fix:**
```bash
# Check source tracking
cat backend/data/source_tracking.json

# If all sources have recent timestamps, wait for next cycle
# If scraping failed, check network and API keys
```

### Issue 2: No Emails Extracted

**Symptoms:**
- Internships discovered but no leads
- Logs show "no_email_found" events

**Causes:**
- Regex not finding emails
- Hunter API not configured
- Hunter daily limit reached

**Fix:**
```bash
# Check Hunter API key
echo $HUNTER_API_KEY

# Check Hunter usage
SELECT hunter_calls FROM daily_usage_stats WHERE date = CURRENT_DATE;

# If limit reached (15), wait for midnight reset
```

### Issue 3: No Drafts Generated

**Symptoms:**
- Leads extracted but no drafts
- Logs show "low_priority" status

**Causes:**
- Full score < 60 (too strict)
- Groq API not configured
- Groq API failing

**Fix:**
```bash
# Check Groq API key
echo $GROQ_API_KEY

# Check scoring
SELECT full_score, COUNT(*) FROM internships 
WHERE full_score IS NOT NULL 
GROUP BY full_score 
ORDER BY full_score DESC;

# If all scores < 60, adjust scoring weights
```

### Issue 4: No SMS Sent

**Symptoms:**
- Drafts generated but no SMS
- Logs show "approval_skipped_no_twilio"

**Causes:**
- Twilio not configured
- Twilio credentials invalid
- SMS daily limit reached

**Fix:**
```bash
# Check Twilio config
echo $TWILIO_ACCOUNT_SID
echo $TWILIO_AUTH_TOKEN
echo $TWILIO_FROM_NUMBER
echo $APPROVER_TO_NUMBER

# Check SMS usage
SELECT twilio_sms_sent FROM daily_usage_stats WHERE date = CURRENT_DATE;

# If limit reached (15), wait for midnight reset
```

### Issue 5: No Emails Sent

**Symptoms:**
- Drafts auto-approved but not sent
- Logs show "Daily email limit reached"

**Causes:**
- Gmail not configured
- Gmail token expired
- Email daily limit reached
- Spacing not met

**Fix:**
```bash
# Check Gmail config
ls -la secrets/gmail_token.json
ls -la secrets/gmail_oauth_client.json

# Check email usage
SELECT emails_sent FROM daily_usage_stats WHERE date = CURRENT_DATE;

# If limit reached (15), wait for midnight reset

# Check spacing
SELECT * FROM check_email_spacing(45);
```
