## 🎯 What Each Panel Shows

### 🔍 Discovery Panel (Top Left)

**Purpose:** Monitor job discovery and scraping performance

**Key Metrics:**
- Internships discovered today/week
- Scrape tier success rates (which method works best)
- Pre-score kill rate (how many get filtered out)
- Firecrawl usage (expensive API calls)

**What to Watch:**
- ⚠️ If Tier 1 success < 80% → scraping issues
- ⚠️ If kill rate > 60% → scoring too strict
- ⚠️ If Firecrawl > 8/10 → approaching limit

---

### 📧 Email Panel (Top Right)

**Purpose:** Track email extraction and validation

**Key Metrics:**
- Total emails found
- Regex (free) vs Hunter (paid) split
- Hunter API usage (15/day limit)
- Validation failure breakdown

**What to Watch:**
- ✅ Higher regex % = saving money
- ⚠️ Hunter > 12/15 → approaching limit
- ⚠️ High validation failures → bad sources

---

### 🚀 Outreach Panel (Bottom Left)

**Purpose:** Monitor email sending and approvals

**Key Metrics:**
- Drafts generated by Groq
- Approval rate (manual + auto)
- Auto-approvals count
- Emails sent today (15/day limit)
- SMS sent today (15/day limit)
- Warmup phase progress
- Pending follow-ups

**What to Watch:**
- ⚠️ Emails 15/15 → daily limit reached
- ⚠️ SMS 15/15 → daily limit reached
- ℹ️ Warmup < 100% → still ramping up
- ✅ Auto-approvals working → hands-free operation

---

### 📊 Performance Panel (Bottom Right, Spans 2 Columns)

**Purpose:** Analyze overall pipeline performance

**Key Metrics:**
- Overall reply rate
- Positive reply rate
- Top company types by reply rate
- Complete pipeline funnel with drop-offs
- Overall conversion rate

**What to Watch:**
- ✅ Reply rate > 10% → excellent
- ⚠️ Big drop-off at "Email found" → need better sources
- ⚠️ Big drop-off at "Validated" → bad email quality
- ⚠️ Big drop-off at "Drafted" → scoring too strict
